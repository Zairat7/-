# -# Movie Library App
**Автор:** [Сабанова Фатимат]

## Описание
Приложение для каталогизации фильмов. Позволяет добавлять записи, хранить их в формате JSON и фильтровать список по жанру или году.

## Функционал
- Валидация: рейтинг только 0-10, год — целое число.
- Сохранение: данные автоматически пишутся в `movies.json`.
- Фильтрация: мгновенный поиск по критериям.

## Пример использования (Тесты)
1. Введите "Inception", "Sci-Fi", "2010", "8.8" -> Нажмите добавить.
2. Введите в поле фильтра "2010" -> В таблице должен остаться только этот фильм.
3. Попробуйте ввести рейтинг "15" -> Приложение выдаст ошибку валидации.
